이차원 배열 행과 열 바꾸기

Nov 22, 2021
#algorithm

Python zip

파이썬의 내장 함수 중 zip을 이용한다면 행열을 쉽게 바꾸어 줄 수 있습니다. 이러한 방식을 자바스크립트에서도 사용해 보겠습니다.

const zip = (r) => r[0].map((_, c) => r.map((r) => r[c]))
